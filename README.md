色彩検定の勉強の一環として色を見分ける力をつけるためのクイズアプリ。

http://angelbell.jp/color-quiz/ に配置しています。http でごめんて。

# 動かし方

React のテンプレそのままです。

## 初回

```
npm install
```

## 起動

```
npm run start
```

## ビルド

```
npm run build
```

## テスト

```
npm run test
```
